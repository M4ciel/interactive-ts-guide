## About The Project

The **Interactive TypeScript Guide** is a dynamic, hands-on learning platform designed to showcase TypeScript concepts, algorithms, and design patterns. Instead of static code blocks, this project uses Vite's raw import capabilities to dynamically fetch and render real, executable `.ts` files directly into the UI.

<!-- GETTING STARTED -->
## Getting Started

### Prerequisites
* npm
    ```bash
    npm install npm@latest-g
    ```

### Installation

1. Clone the repo
    ```bash
    git clone https://github.com/M4ciel/interactive-ts-guide.git
    ```
2. Enter in the project folder
    ```bash
    cd interactive-ts-guide
    ```
3. Install NPM packages
    ```bash
    npm install
    ```
4. Run the development server
    ```bash
    npm run dev
    ```
<p align="right">(<a href="#readme-top">back to top</a>)</p>


<!-- FEATURES -->
## Features
* ⚡ **Dynamic Code Rendering:** Fetches real `.ts` files using Vite's `?raw` import and renders them with syntax highlighting.
* 🎨 **Modern UI/UX:** Built with Tailwind CSS and Shadcn UI components for a premium, accessible dark-mode experience.
* 🗂️ **Categorized Snippets:** Easy navigation between Algorithms (Sorts, Searches), Data Structures, and Design Patterns.
